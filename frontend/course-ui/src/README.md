# FinCourse Explorer

## Description
FinCourse Explorer is a Course Search Application built using React and Spring Boot.

Users can search for LMS-related courses and view course details dynamically.

## Technologies Used
- React JS
- Spring Boot
- Java
- REST API

## Features
- Course Search
- Dynamic Course Display
- Frontend and Backend Integration
- JSON API Response
- Static Course Data

## Backend Run Steps
1. Open backend/courseapp
2. Run:
mvn spring-boot:run

Backend URL:
http://localhost:8080

## Frontend Run Steps
1. Open frontend/course-ui
2. Run:
npm start

Frontend URL:
http://localhost:3000

## Example API
GET:
http://localhost:8080/api/courses/search?keyword=react
