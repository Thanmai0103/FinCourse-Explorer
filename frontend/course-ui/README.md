# FinCourse Explorer

FinCourse Explorer is a Finchtalk LMS Course Search Application developed using React JS and Spring Boot.

The application allows users to search LMS-related courses dynamically and display matching course information through frontend-backend integration.

---

## Technologies Used

### Frontend
- React JS
- Axios
- CSS

### Backend
- Spring Boot
- Java

### Database
- Static Course Data (No database used)

---

## Features

- Dynamic Course Search
- Frontend and Backend Integration
- REST API Implementation
- JSON Response Handling
- Modern Responsive UI
- Animated Gradient Design
